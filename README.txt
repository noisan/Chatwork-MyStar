Chatwork MyStar
====
[チャットワーク](chatwork.com)で非公式のスター機能を使えるようにするChromeエクステンションです。

公式スター機能が実装されるまでのつなぎとして自分用に作っているものを公開しています。
あなたのお役に立てればとても嬉しいです。

このエクステンションは非公式機能のため、
公式サイトの仕様変更により突然動かなくなることがあります。


インストール方法
----
リポジトリを`git clone`するか、zipファイルをダウンロードしてください。

以下のファイルを置く専用ディレクトリを作ってそこに保存します：

* manifest.json
* script.js

そして、Chromeの「設定」→「拡張機能」の画面を開き、
この2つのファイルを入れたフォルダをドラッグ＆ドロップします。


「非公式スター機能」の使い方
----
このエクステンションは、チャットワークの画面に次の2つの機能を追加します：

* 各チャットメッセージのタイムスタンプ右側に「☆」ボタン
  - スターのon/off

* ページヘッダのチャットグループ名のピンの右側に「☆スター付き」ボタン
  - スター付きメッセージの表示切り換え
